# Prompt Engineering Content

This directory contains structured content for the "Mastering Prompt Engineering" seminar by Syed Monowarul Islam.

## Structure

The content is organized following the seminar's four main sections:

### 1. Foundation & Understanding
- What Is Prompt Engineering?
- Why Prompt Engineering Matters
- When AI Gets It Wrong
- Understanding LLM Limitations
- How LLMs Think: The Token Game

### 2. Core Components
- Standard Prompts: The Starting Point
- System Messages: The Hidden Guide
- Steering AI with System Messages

### 3. Context Mastery
- Context: The Power of Specificity
- Context Windows & Token Limits
- Lost in the Middle: Context Placement Matters
- Context Best Practices

### 4. Advanced Techniques
- Personas & Roles: Unlocking Expertise
- Crafting Effective Personas
- The Complete Prompt Framework
- Common Prompt Patterns
- Building Your Prompt Evolution

## File Structure

Each topic will have corresponding files in the following directories:
- `cheatsheet/` - Quick reference guides and key takeaways
- `lessons/` - Detailed explanations and learning materials
- `examples/` - Practical examples and use cases

## Data Format

All content follows the established TypeScript cheatsheet format with:
- `id`: Unique identifier
- `title`: Topic title
- `explanation`: Overview and context
- `codeBlocks`: Practical examples with code
- `tip`: Key takeaways and best practices
